\section{Заключение}

В заключение приведем краткую сводку результатов полученных в данной
главе: 
\begin{itemize}
\item Поляризация среды - дипольный момент единицы объема является
  реакцией среды на внешнее электромагнитное воздействие.
\item Поляризация среды может быть представлена в виде ряда по
  внешнему полю. 
\item Существует два подхода к описанию нелинейных восприимчивостей:
  феноменологический (общий подход) и расмотрение конкретных можельных
  систем. 
\item При рассмотрении модели электрона в некоторой потенциальной яме
  нами был получен важный результат: линейный и квадратичный отклики
  среды имеют одинаковый порядок когда внешнее воздействие имеет
  порядок внутриатомного поля
\item Существует особый класс сред - фоторефрактивные кристаллы в
  которых реакция среды на внешнее электромагнитное поле обусловлена
  электрооптическим эффектом на внутренних электрических полях. 
  В результате время отклика в этом случае намного больше чем когда
  реакция среды обусловлена нелинейным откликом отдельного атома или
  молекулы. 
\end{itemize}

В качестве материала для этой главы (кроме тех источников что указаны
прямо в тексте) использовались 
\begin{itemize}
\item \cite{bMSULecture} - для всего кроме фоторефрактивных материалов
\item \cite{bMacswell} - феноменологическое описание восприимчивостей и
  нелинейная восприимчивость газа свободных электронов
\item \cite{bPetrov, bMyDiser} - фоторефрактивные материалы
\end{itemize}
