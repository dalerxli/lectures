\section{Заключение}

В заключение приведем краткую сводку результатов полученных в данной
главе: 
\begin{itemize}
\item Дифракция света в ФРК может быть описана с помощью метода
  интегральных уравнений
\item Фенинг - особый тип двухволнового рассеяния света в ФРК
\item Схема двойного некогерентного обращающего зеркала (ДНОЗ) -
  базовая схема для обращения волнового фронта в ФРК
\item Процесс генерации обращенных световых пучков в ДНОЗ имеет
  пороговый характер 
\end{itemize}

В качестве материала для этой главы (кроме тех источников что указаны
прямо в тексте) использовались 
\begin{itemize}

\item \cite{bPetrov} - дополнительный материал
\end{itemize}
